# 🐍 Django Docker Application

A Dockerized Django web application developed as part of a Cloud Computing coursework assignment. This project demonstrates how to containerize a Django application using Docker and Docker Compose for simplified deployment and development.

---

## 📖 Project Overview

This project showcases the deployment of a Django web application inside Docker containers. Docker Compose is used to simplify container management and ensure a consistent development environment.

---

## ✨ Features

- Django Web Framework
- Docker Containerization
- Docker Compose Support
- Lightweight Python Environment
- Development Ready
- Easy Deployment

## 🚀 Running the Project

### Build Docker Image

```bash
docker-compose build
```

### Start Container

```bash
docker-compose up
```

Visit

```
http://localhost:8000
```

---

## Stop Container

```bash
docker-compose down
```

---

## Rebuild

```bash
docker-compose up --build
```
